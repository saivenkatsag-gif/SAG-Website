Settings & Database

🗄️ Supabase Database Connection

Connect to your Supabase project for real-time data persistence. Applications will be stored in a drone_applications table and sync live across all sessions.

📋 Supabase Table Setup SQL

Run this SQL in your Supabase SQL Editor to create the required table:

CREATE TABLE IF NOT EXISTS drone_applications (
  id BIGSERIAL PRIMARY KEY,
  emoji TEXT NOT NULL DEFAULT '🚁',
  tab_label TEXT,
  title TEXT NOT NULL,
  title_highlight TEXT DEFAULT '',
  description TEXT NOT NULL,
  cta_text TEXT DEFAULT 'Enquire →',
  cta_link TEXT DEFAULT 'tel:+918977776019',
  badge TEXT DEFAULT '',
  status TEXT DEFAULT 'active',
  accent_color TEXT DEFAULT '#2ecc71',
  image_url TEXT DEFAULT '',
  features JSONB DEFAULT '[]'::jsonb,
  display_order INTEGER DEFAULT 0,
  created_at TIMESTAMPTZ DEFAULT NOW(),
  updated_at TIMESTAMPTZ DEFAULT NOW()
);

-- Enable Row Level Security (allow public read, authenticated write)
ALTER TABLE drone_applications ENABLE ROW LEVEL SECURITY;

CREATE POLICY "Public read" ON drone_applications FOR SELECT USING (true);
CREATE POLICY "Auth write" ON drone_applications FOR ALL USING (true);

-- Enable realtime
ALTER PUBLICATION supabase_realtime ADD TABLE drone_applications;

🏷️ Categories Table Setup SQL

Run this SQL to create the categories table:

CREATE TABLE IF NOT EXISTS categories (
  id BIGSERIAL PRIMARY KEY,
  name TEXT NOT NULL UNIQUE,
  description TEXT DEFAULT '',
  created_at TIMESTAMPTZ DEFAULT NOW(),
  updated_at TIMESTAMPTZ DEFAULT NOW()
);
ALTER TABLE categories ENABLE ROW LEVEL SECURITY;
CREATE POLICY "Public read" ON categories FOR SELECT USING (true);
CREATE POLICY "Auth write" ON categories FOR ALL USING (true);

📋 Full Drones Table Setup SQL

Also run this SQL to create the full_drones table for the Full Drones section:

CREATE TABLE IF NOT EXISTS full_drones (
  id BIGSERIAL PRIMARY KEY,
  name TEXT NOT NULL,
  price NUMERIC NOT NULL DEFAULT 0,
  original_price NUMERIC,
  tagline TEXT DEFAULT '',
  image TEXT DEFAULT '',
  brochure_url TEXT DEFAULT '',
  wa_num TEXT DEFAULT '919390238537',
  is_new BOOLEAN DEFAULT false,
  is_offer BOOLEAN DEFAULT false,
  status TEXT DEFAULT 'instock',
  specifications JSONB DEFAULT '[]'::jsonb,
  created_at TIMESTAMPTZ DEFAULT NOW(),
  updated_at TIMESTAMPTZ DEFAULT NOW()
);

ALTER TABLE full_drones ENABLE ROW LEVEL SECURITY;
CREATE POLICY "Public read" ON full_drones FOR SELECT USING (true);
CREATE POLICY "Auth write" ON full_drones FOR ALL USING (true);
ALTER PUBLICATION supabase_realtime ADD TABLE full_drones;
